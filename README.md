# DIKWP SemanticMath LearnLab 2026 V1

A standalone, offline-first prototype for learning new mathematics needed by future AI systems, including semantic mathematics.

## Core workflow
Learner profile → AI math roadmap → concept/semantic/expression spaces → DIKWP semantic transformations → interactive labs → proof tickets → AI project bank → Evidence Ledger → AI Use Log → SemanticMath Learning Passport.

## Quick start
Open `index.html` in a modern browser. No server, database, API key, theorem-prover server, account, or network is required.

Optional CLI:

```bash
python tools/run_semantic_math_plan.py examples/sample_learner.json --out outputs
```

## Boundary
This is a learning and curriculum prototype. It is not a formal proof certificate, official degree, automated grading system, or mathematical research validation platform. Outputs require teacher, peer, or domain expert review.
